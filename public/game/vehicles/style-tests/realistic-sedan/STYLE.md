# Realistic Sprite Style Reference

Selected direction: Variant B (`selected-style-b-reference.png`).

Use this style for the next gameplay sprite pass:

- realistic real-world proportions, not toy-like
- exact 90-degree overhead top-down view
- clean game-readable materials with restrained detail
- believable paint, glass, headlights, and panel seams
- no brands, logos, license text, road, background, or baked shadow
- shadows should be rendered in code, not inside the PNG
- avoid sci-fi, neon, cyberpunk, capsule shapes, pixel art, and cartoon exaggeration

Variant B is preferred over:

- A/D: more product-render detail than needed for small in-game scale
- C: too muted and less premium
- E: readable, but slightly less convincing as a real car
